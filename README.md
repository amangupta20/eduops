# EDUOPS

An open-source local learning platform that teaches Docker by dropping you
into broken environments and coaching you through fixing them in real time.

No cloud account. No managed sandbox. Just you, your terminal,
and real Docker commands producing real outcomes.

## Status

🚧 Under active development

## Requirements

- Python 3.11+
- Docker (running on host)
